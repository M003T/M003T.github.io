Projet de Vision Artificielle et Reinforcement Learning

d'une IA joueuse de jeu de société en temps réel en Python - Projet en cours
Projet de Vision Artificielle d'une IA joueuse de jeu de société en temps réel en Python
Python IA Vision artificielle Deep learning OpenCv PyTorch
Pour ce projet, l'objectif etait de mettre en place un adversaire contrôlé par Intelligence Artificielle sur le jeu de société physique Mr. Jack Pocket. Après avoir mis en place des technologies de vision artificielle afin de reconnaître les différentes pièces du jeu en utilisant une combinaison de techniques classiques de la vision artficielle et de Deep Learning, nous avons ensuite mis en place un adversaire à l'aide d'une intelligence artificielle basée sur un algorithme MinMax. Nous utilisons pour ce projet OpenCV pour la partie traitement d'image ainsi que PyTorch pour la partie apprentissage

Pilotage Autonome de Drones

Grâce au langage Multi-Agent SARL et le simulateur Airsim de Microsoft sous Unreal Engine
Projet de pilotage autonome de drônes grâce au langage Multi-Agent SARL
Sarl Language Multi-Agents Programmation orientée Agent Simulation Système décisionnel Détection d'obstacle LiDar AirSim Unreal Engine Client API
Le but de ce projet était le pilotage autonome de plusieurs drones en simultané dans un environnement avec obstacles. Nous avons pour cela utilisé le language orienté agent SARL afin de mettre en place un système de décision leur permettant de choisir une destination puis de s'y rendre tout en évitant les obsctacles captés grâce aux capteurs LiDar ainsi que les autres agents drones. La partie simulation du projet à alors été réalisée grâce au simulateur AirSim de Microsoft tournant en temps réel sur Unreal Engine grâce aux APIs fournies

Intelligence Artificielle pour le jeu "Pogo"

Grâce au langage de programmation logique Prolog avec interface graphique en Qt
Projet d'Intelligence Artificielle pour le jeu "Pogo" avec Prolog et Qt
Intelligence artificielle Programmation Logique Prolog C++ Qt
Le but de ce projet était de mettre en place une intelligence artificielle "classique" sur le jeu Pogo. Nous avons pour cela utilisé le language de programmation logique Prolog afin de calculer à partir d'un état initial l'ensemble des états atteignables après un déplacement, de leur donner une valeur d'évaluation afin de savoir laquelle approche le plus l'IA de la victoire puis de l'utiliser pour parcourir l'arbre des solutions avec plusieurs tours d'avance afin de séléctionner le coup optimal. Pour ce projet, l'interface graphique a été réalisée sous C++ à l'aide de Qt

Gestion des réservations d'un court de Tennis

Base de donnée SQL avec Interface Graphique grâce à Microsoft Access et VBA
Projet de Gestion des réservations d'un court de Tennis avec Base de données SQL et interface graphique Microsoft Access + VBA
Base de données SQL Access VBA Gestion utilisateurs Gestion accès Application métier
Le but de ce projet était de réaliser la gestion complète d'un club de tennis pour un client, de la gestion des membres avec leur inscription et leur cotisation à la gestions des cours et des moniteurs, des réservations des terrains dans le cadre des cours ou encore pour l'organisation de compétitions en équipe. Nous avons dû mettre en place pour cela une base de donnée complète sur Microsoft Access et nous avons pu réaliser de la même manière une interface graphique en utilisant VBA
